VemLang Programming Language
Overview
VemLang is an open‑source interpreted programming language written in C on December 30, 2025.
It was created to fulfill the author’s greatest aspiration: to create a programming language.
Although VemLang was initially developed for enjoyment, it is fully functional and can be used normally.
